# How to run this projects

First clone this repository.

```bash
   $ git clone https://github.com/sirotros/twitter-clone.git
```

Go to folder

```bash
  cd twitter-clone
```

Install dependencies. Make sure you already have nodejs & npm installed in your system.

```bash
   $ npm install
```

Run it

```bash
   $ npm start
```
